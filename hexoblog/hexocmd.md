hexo常用命令

```bash
hexo init folder
cd folder
npm install
```

新建文章

```bash
hexo new [layout] <title>
#example
hexo new post "今日好心情"
hexo new page --path about/me
```

生成文章

```bash
hexo generate
```

其他实用命令

```bash
#启动hexo服务
hexo server
#清除hexo生成
hexo clean
#查看列表
hexo lsit <type>
#查看版本
hexo version
```
